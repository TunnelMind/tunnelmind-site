ATAP application profiles

ATAP — the Agent Trust Attestation Protocol — is a structural envelope. An application profile fills it with the domain-specific vocabulary for one class of agent: the witness event types, the payload JSON Schemas, the aggregation rules, and the contents of the receipt's profile artifacts (ATAP v0.1 §9).

The source of truth is github.com/TunnelMind/atap-profiles. New profiles are added by pull request; CI runs the §9.4 acceptance checks and auto-merges a passing PR within 7 days.
